Instrucciones de compilación:
----------------------------

Obtener las bibliotecas necesarias (utilizando el gestor de paquetes - apt, yum,...):

libsdl2-dev, libsdl2-image-dev, libsdl-mixer-dev, libogg-dev, libvorbis-dev, mesa-common-dev, libz-dev

Descarga el código de:

http://github.com/AugustoRuiz/UWOL

En la carpeta UWOL ejecutar:

./makelinux.sh

Se creará el ejecutable en la carpeta bin.

Build instructions:
------------------

Get the following libraries:

libsdl2-dev, libsdl2-image-dev, libsdl-mixer-dev, libogg-dev, libvorbis-dev, mesa-common-dev, libz-dev

Download the code from: 

http://github.com/AugustoRuiz/UWOL

In UWOL folder, run:

./makelinux.sh

The executable will be created in bin folder.
